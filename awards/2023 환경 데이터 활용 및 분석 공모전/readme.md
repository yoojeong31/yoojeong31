## 2023 환경 데이터 활용 및 분석 공모전 | 우수상(분석 아이디어 부문)

---

## 개요
- 주최/주관 : 환경부
- 기간 : 2023.04.24 – 2023.06.04
- 팀 구성 : Nature Republic (2명)
- 수상 : 우수상
- 한 줄 소개 : 기존 다중이용시설의 공기질과 이용 데이터를 활용해 신축 예정 시설의 실내공기질을 사전 예측하고, 이상 데이터를 검출하는 분석 아이디어 제안

---

## 아이디어 요약
- 배경/문제
  - 실내 생활 비중이 커지고 대규모 다중이용시설이 늘면서, 개장 이후 대응이 아니라 개장 전 공기질 리스크를 예측할 필요가 있음
  - 특히 신축 시설은 측정 데이터가 없어, 유사 환경/운영 특성을 가진 시설 데이터를 학습해 사전 예측하는 접근이 중요함
    
- 목표 : 신축 예정 다중이용시설의 실내공기질을 사전 예측하고, 공기질이 현저히 나쁜/좋은 상태(이상 데이터)를 식별
  
- 활용 데이터(공공)

- 접근 방법 : 유사 환경 시설을 군집화한 뒤 이상치 탐지로 특이 케이스를 추출하고 결과를 시각화
  
- 기대효과
  - 예방 중심 관리 : 신축 확정 단계부터 공기질을 예측해 사전 예방 및 조치 가능
  - 투명성/신뢰 : 예측 결과를 사전에 공개해 이용자에게 정보 투명성 제공 가능
  - 정책/운영 의사결정 지원 : 예측 데이터를 기반으로 시설 설계·환기·관리 대책 수립에 참고 가능

---

## 내가 수행한 역할
- 분석 아이디어 기획부터 자료 조사, 데이터 구성 정의, 모델 흐름 정리까지 전 과정에 참여
- 팀 산출물 중 발표자료 제작 비중을 크게 담당
- 결과를 문서화하고 제출 형식에 맞게 내용 정리(국민평가 요약자료 포함)

---

## 성과
- 실내공기질을 사후 조치가 아닌 사전 예측 관점으로 확장한 분석 아이디어로 경쟁력 인정
- 군집화 + 이상치 탐지 조합으로 유사 환경군 대비 공기질이 특이한 시설을 선별하는 방향 제시
